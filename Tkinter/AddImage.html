Add Image

The code is to add the image “kakashi.jpg”. Variable “width” and “height” have the height and width of the image. 
from PIL import Image, ImageTk

ImageFile='kakashi.jpg'
load = Image.open(ImageFile)
render = ImageTk.PhotoImage(load)
img = Label(root, image=render)
img.image = render
img.place(x=0, y=0)
width, height = load.size
